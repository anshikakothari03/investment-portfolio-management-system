Description:

I developed a web-based Investment Portfolio System that allows interaction with five database with tables: USERS, ASSETS, PORTFOLIOS, HOLDINGS, and TRANSACTIONS.

The application supports:
- Viewing all tables using navigation links
- Inserting new records through forms
- Updating existing user details (name, email, role, status)
- Deleting records dynamically
- Searching users by name

All tables are displayed in structured format with proper headers, and the system is implemented using PHP and MySQL on the Betaweb server.
