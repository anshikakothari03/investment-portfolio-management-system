# Investment Portfolio Management System

## Overview

A web-based investment portfolio management system built using MySQL, SQL, PHP, HTML, and CSS.

The application allows users to manage portfolios, assets, holdings, and transactions through a relational database architecture with full CRUD functionality.

---

## Features

- User Management
- Asset Management
- Portfolio Tracking
- Holdings Monitoring
- Transaction Management
- Search Functionality
- CRUD Operations
- Role-Based Interfaces

---

## Database Design

The system consists of five relational tables:

1. USERS
2. ASSETS
3. PORTFOLIOS
4. HOLDINGS
5. TRANSACTIONS

The schema was normalized to BCNF to eliminate redundancy and ensure data integrity.
